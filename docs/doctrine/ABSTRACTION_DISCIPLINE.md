# ABSTRACTION DISCIPLINE

- orchestrator fokus pada direction & decomposition => MANAGER_ORCHESTRATOR
- specialist/ architect fokus pada domain  => UX_SPECIALIST, UX_RETENTION, ARCHITECTURE_GUARDIAN, BACKEND_SPECIALIST
- executor fokus pada implementasi => BACKEND_EXECUTOR, FRONTEND_EXECUTOR

- hindari terlalu cepat masuk implementation detail
- jangan menentukan micro-execution jika belum diperlukan
- prioritaskan strategic clarity over detailed control

Rule:
semakin tinggi role agent,
semakin tinggi level abstraction yang harus dijaga.
